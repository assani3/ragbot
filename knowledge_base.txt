### Project: RAG Portfolio Chatbot
This project. A Retrieval-Augmented Generation (RAG) chatbot built with the
OpenAI API that answers questions about Assani's background and projects. It
works by splitting this knowledge base into chunks, generating embeddings for
each chunk using OpenAI's embedding model, and storing those embeddings in
memory. When a user asks a question, the question is embedded and compared
against the stored chunk embeddings using cosine similarity to retrieve the most
relevant context. That context is then passed to a chat completion call so the
model answers using grounded, relevant information instead of guessing. This
demonstrates practical experience with LLM application development, embeddings,
and RAG pipeline design.
